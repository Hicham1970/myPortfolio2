A Blog app built using Next.js 15, MDX (markdown), Prismjs and Tailwind CSS,. This app is static site that renders the markdown posts..


## Technologies

- **NextJs**: A React framework for building static and dynamic websites.
- **mdx-js/loader**: This package is a webpack loader to support MDX.js files in your project.
- **Prismjs**: A lightweight, extensible syntax highlighter.
- **mdx-js/react**: This package provides a way to use MDX in React applications.   
- **grey-matter**: A library for parsing front matter from strings or files.
- **next-mdx-remote**: This package allows you to load MDX content from remote sources in Next.js applications.
- **Tailwind CSS**: A utility-first CSS framework for building custom designs.
- **Vercel**: A cloud platform for static sites and serverless functions.

## Getting Started

To get started with this project, you can clone the repository and install the
dependencies:

```bash
git clone
cd hugoblog
npm install
```

Once the dependencies are installed, you can run the development server:

```bash
npm run dev
```

The development server will start at `http://localhost:3000`.

## Deployment

To deploy this project, you can use Vercel. You can link your GitHub repository
to Vercel and deploy the project with a single click.

This blog site is a great example of how you can build a full-stack
